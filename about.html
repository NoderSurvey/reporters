关于我们

简介


我们是以 Ecma 成员的名义发起的一个 Node.js 开发者报告统计组织。我们的成员来自多个机构，包括 Ecma、TC39、Node.js 基金会等。同时也包含一系列来自阿里巴巴和腾讯的公司。


动机


Node.js 开发者报告的诞生源自于改善国内 Node.js 开发者生态的现状。基于这一点，我们在 2020 年第一次发起了 Node.js 开发者报告调查。


第一次调查的目的是为了系统的了解国内 Node.js 生态的现状，根据第一年的报告数据，后期组织会调整问卷内容以及统计方式。问卷内容后面会 host 在 Github 开源组织上由组织成员日常运营，欢迎社区同学来 PR。
